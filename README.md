
A Streamlit coursework project for uploading, cleaning, visualizing, and exporting tabular datasets.


1. Open the project folder in a terminal.
2. Create a virtual environment: python -m venv .venv
3. Activate it: .venv\\Scripts\\activate
4. Install dependencies: `pip install -r requirements.txt
5. Start the app: python -m streamlit run app.py

If the project folder was moved, delete .venv, create it again, and reinstall the requirements.

Important files and folders:

app.py - main Streamlit entry point
home.py - Home page
pages/ - coursework pages
utils/ - shared helpers
sample_data/ - bundled datasets
requirements.txt - Python dependencies
